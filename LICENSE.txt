--------------

1.1. "Contributeur"
    signifie chaque individu ou entité juridique qui crée, contribue à

1.2. "Version contributeur"
    signifie la combinaison des contributions d'autrui ( si ) utilisé
    par un contributeur et la contribution de ce contributeur particulier.

1.3. "Contribution"
    signifie logiciel couvert d'un contributeur particulier.

1.4. "Logiciel couvert"
    signifie le formulaire de code source auquel le contributeur initial a joint
    l'avis de la pièce A, le formulaire exécutable de ce code source
    Formulaire et modifications de ce formulaire de code source, dans chaque cas
    y compris des parties de celui-ci.

    ( a ) que le contributeur initial a joint l'avis décrit

        version 1.1 ou antérieure de la licence, mais pas également sous le
        termes d'une licence secondaire.

1.6. "Formulaire exécutable"
    désigne toute forme de travail autre que le formulaire de code source.

1.7. "Travail plus important"
    signifie un travail qui combine le logiciel couvert avec d'autres matériaux, en
    un ou plusieurs fichiers distincts, qui ne sont pas des logiciels couverts.

1.8. "Licence"
    signifie ce document.

1.9. "Licensable"
    signifie avoir le droit d'accorder, dans toute la mesure du possible,
    que ce soit au moment de la subvention initiale ou ultérieurement,
    tous les droits transmis par cette licence.

1.10. "Modifications"
    désigne l'un des éléments suivants:

    ( a ) tout fichier sous forme de code source résultant d'un ajout à,
        suppression ou modification du contenu de Covered
        Logiciel; ou

    ( b ) tout nouveau fichier dans le formulaire de code source qui contient tout couvert
        Logiciel.

1.11. "Créances de brevets" d'un contributeur
    désigne toute revendication de brevet ( s ), y compris, sans limitation, la méthode,

    Contributeur qui serait violé, mais pour l'octroi de la

    signifie la forme du travail préféré pour apporter des modifications.

1.14. "Vous" ( ou "Votre" )
